# My Portfolio Website

This repository contains a simple portfolio website that I created for
the **User Interface Design (23CSE113)** lab project.

The aim of this project was to understand how basic webpages are built
using **HTML and CSS** and how different sections of a website are
organized.

## About the Website

The portfolio is a basic multi‑section webpage that introduces me and
shows some example work.\
I tried to keep the design simple and clean so that it is easy to read
and navigate.

The website includes the following sections:

-   **Home** -- A welcome message and a short introduction.
-   **About** -- A brief description about me and my interests.
-   **Projects** -- Some sample project ideas with images and short
    explanations.
-   **Contact** -- Basic contact details.
-   **Footer** -- A small footer section at the bottom of the page.

## Technologies Used

The following technologies were used to build this website:

-   **HTML** for creating the structure of the webpage
-   **CSS** for styling and layout
-   **GitHub Pages** for hosting the website online

## Responsive Design

A simple **CSS media query** is used so that the website layout adjusts
on smaller screens such as mobile devices. This helps the project remain
readable on different screen sizes.

## How to View the Website

The project is hosted using **GitHub Pages**.\
After uploading the files to GitHub and enabling Pages in the repository
settings, the website can be accessed through the generated GitHub Pages
link.
